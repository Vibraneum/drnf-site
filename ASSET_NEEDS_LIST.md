- Keep same filename OR update src attribute
- Maintain alt text (update if needed)

**Step 3: Optimize New Image**
- Compress to web-friendly size
- Convert to WebP if possible
- Maintain aspect ratio

**Step 4: Test**
- Check loading speed
- Verify responsive behavior
- Ensure accessibility

## üìù IMAGE SPECIFICATIONS

### Standard Sizes:
- **Hero banners:** 1920x800px (2.4:1 ratio)
- **Section headers:** 1200x600px (2:1 ratio)
- **Card backgrounds:** 600x400px (3:2 ratio)
- **Portraits:** 400x400px (1:1 ratio)
- **Thumbnails:** 300x300px (1:1 ratio)

### File Requirements:
- **Format:** JPG for photos, PNG for transparency, WebP for modern browsers
- **Size:** < 200KB per image (optimized)
- **Quality:** 80-85% compression (good quality, small size)
- **Naming:** Descriptive (loocafe-hero.jpg, chai-counter-action.jpg)

## üé® DESIGN NOTES

### Image Treatment:
- **Overlays:** Subtle gradient overlays on hero images for text readability
- **Filters:** Slight warmth added (+5% saturation) for vibrant feel
- **Borders:** Rounded corners (24px) to match site design
- **Shadows:** Drop shadows on cards (0 8px 32px rgba(102,126,234,0.15))

### Responsive Behavior:
- **Desktop:** Full resolution images
- **Tablet:** Medium resolution (800px width)
- **Mobile:** Small resolution (600px width) or cropped versions
- **Lazy loading:** All images below fold use lazy loading
